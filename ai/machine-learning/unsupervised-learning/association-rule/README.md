## Goal
가게 주인이 가게 매출을 올리고자 각 품목에 대해 1+! 행사를 시행하고자 한다. 각 품목에 대하여 어떤 품목이 가장 소비자 효과가 크게 나타날 선택일까?

## Note
지지도 분석만 필요로 하는 상황이라면 더 속도가 빠른 Eclat을 사용하고, 나머지 상황들에 대하여 Apriori를 사용한다.
